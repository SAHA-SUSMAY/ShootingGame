==============================
🎮 シューティングゲーム
==============================

📌 ゲームの説明
------------------------------
このゲームは、プレイヤーが敵を撃って倒す2Dシューティングゲームです。
ステージをクリアしながらスコアをためていきましょう！

🕹️ 操作方法
------------------------------
← → キー： プレイヤーの左右移動  
スペースキー： 弾を撃つ  
Enterキー： ステージクリア後やゲームオーバー後に次へ進む  

🎯 ルール
------------------------------
- プレイヤーは画面下にいて、上からくる敵を撃ちます。
- 弾が敵に当たると、敵のライフが減ります。
- 敵のライフが0になると倒せます。
- 敵がすべていなくなるとステージクリアです。
- プレイヤーが敵に当たるとゲームオーバーになります。
- Enterキーを押すと、次のステージに進むか、ゲームをやり直せます。

🏆 スコアとステージ
------------------------------
- 敵を倒すとスコアが増えます。
- ステージが進むと敵が強くなります。


📂 ファイル構成
------------------------------
- GameFrame.java： ゲームのメイン処理
- GameWorld.java： ゲームの状態（プレイヤー、敵、スコアなど）
- Player.java： プレイヤーの動きと描画
- Enemy.java / EnemyBase.java： 敵の動きとライフ
- PlayerBullet.java： プレイヤーの弾の処理
